# AI-Driven Secure Communication Protocols for 6G Networks Using Federated Learning and Lightweight Encryption

## Overview

This project presents an intelligent and secure communication framework for 6G Massive Internet of Things (mIoT) networks using Federated Learning, FT-Transformer, RUN Optimization, and Ascon-128 Lightweight Encryption.

The framework focuses on privacy-preserving distributed learning, binary cyberattack detection, and secure communication in large-scale 6G environments.

## Problem Statement

Current 6G communication systems face several challenges:

* Non-IID data distribution
* Client heterogeneity
* Privacy preservation
* Communication overhead
* Accurate cyberattack detection
* Secure data transmission

This project addresses these challenges through a RUN-Optimized FedProx-Enabled FT-Transformer architecture integrated with Ascon-128 lightweight encryption.

## Objectives

* Develop a secure communication framework for 6G mIoT networks.
* Detect malicious network traffic using AI-based attack detection.
* Improve federated learning performance in non-IID environments.
* Ensure privacy-preserving distributed model training.
* Secure legitimate communication using lightweight encryption.
